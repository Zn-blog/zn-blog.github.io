# 🐑 羊了个羊 - 消除游戏

## 🎮 游戏简介

这是一个类似"羊了个羊"的消除类小游戏。通过点击相同的卡片进行消除，挑战你的观察力和策略能力！

## 🎯 游戏规则

### 基本玩法
1. **点击卡片**：点击游戏板上的卡片，卡片会移动到下方的槽位
2. **消除规则**：当槽位中有3个相同的卡片时，它们会自动消除
3. **胜利条件**：消除所有卡片即可过关
4. **失败条件**：槽位被填满（7个）且无法消除时游戏失败

### 特殊规则
- **卡片遮挡**：被其他卡片遮挡的卡片无法点击（会抖动提示）
- **层级系统**：卡片分为3层，上层卡片会遮挡下层卡片
- **关卡递增**：每过一关，卡片数量和种类都会增加

## 🎁 道具系统

### 洗牌 (3次)
- 重新随机排列游戏板上的所有卡片
- 可以帮助你找到被遮挡的卡片
- 每局游戏有3次使用机会

### 撤销 (3次)
- 撤销上一步操作
- 可以恢复误点的卡片
- 每局游戏有3次使用机会

## 📊 计分系统

- **消除得分**：每消除3张卡片得30分
- **过关奖励**：每过一关额外获得100分
- **关卡难度**：关卡越高，卡片越多，挑战越大

## 💡 游戏技巧

### 新手技巧
1. **优先消除上层**：先点击上层卡片，避免遮挡
2. **观察全局**：在点击前观察所有可见卡片
3. **规划槽位**：注意槽位数量，避免填满
4. **善用道具**：在关键时刻使用洗牌和撤销

### 进阶技巧
1. **记忆位置**：记住相同卡片的位置
2. **预判消除**：提前规划消除顺序
3. **保留空间**：不要让槽位太满
4. **战略撤销**：在即将失败时使用撤销
