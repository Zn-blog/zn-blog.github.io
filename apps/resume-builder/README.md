# 简历生成器

一个功能强大的在线简历生成工具，支持多种专业模板、照片上传、数据管理和多格式导出。

## 🌟 主要特性

### 📝 智能表单系统
- **多页简历设计** - 不限于单页布局，支持完整的多页简历
- **动态表单** - 可添加多个教育背景、工作经历和项目经验
- **实时验证** - 智能表单验证，确保数据完整性
- **自动保存** - 防止数据丢失的自动保存功能

### 📷 照片上传功能
- **拖拽上传** - 支持拖拽和点击上传个人照片
- **格式支持** - 支持 JPG、PNG 等常见图片格式
- **尺寸优化** - 自动优化照片尺寸和质量
- **预览功能** - 实时预览照片效果

### 🎨 多样化模板
1. **经典商务** - 传统正式风格，适合商务岗位
2. **现代简约** - 简洁时尚设计，适合创意岗位
3. **创意设计** - 个性创新布局，适合设计师
4. **技术极客** - 代码风格界面，适合程序员
5. **学术研究** - 严谨专业格式，适合研究员

### 💾 数据管理系统
- **JSON存储** - 所有数据保存为JSON格式，便于编辑和管理
- **简历管理** - 创建、编辑、删除多份简历
- **版本控制** - 记录创建和更新时间
- **数据导入导出** - 支持数据的备份和恢复

### 📤 多格式导出
- **PDF导出** - 高质量PDF文件，支持多页内容
- **Excel导出** - 结构化数据表格，便于HR筛选
- **打印优化** - 针对打印进行的样式优化

## 🚀 快速开始

### 1. 基本信息填写
- 填写姓名、联系方式等必填信息
- 上传个人照片（可选）
- 输入求职意向和基本资料

### 2. 添加详细信息
- **教育背景** - 点击"添加"按钮添加学历信息
- **工作经历** - 详细描述工作经验和职责
- **项目经验** - 展示重要项目和技术栈
- **技能特长** - 列出专业技能和特长
- **自我评价** - 简要介绍个人优势

### 3. 选择模板
- 在底部模板区域选择合适的样式
- 实时预览不同模板效果
- 根据求职岗位选择最佳模板

### 4. 保存和导出
- 点击"保存简历"创建简历档案
- 使用"导出PDF"生成打印版本
- 使用"导出Excel"生成数据表格

## 🎯 使用技巧

### 模板选择建议
- **商务岗位** → 经典商务模板
- **创意岗位** → 现代简约或创意设计模板
- **技术岗位** → 技术极客模板
- **学术研究** → 学术研究模板

### 内容填写建议
1. **简洁明了** - 避免冗长描述，突出重点
2. **量化成果** - 用数字说话，展示具体成就
3. **关键词优化** - 包含行业相关关键词
4. **逻辑清晰** - 按时间倒序排列经历

### 照片要求
- **尺寸建议** - 200x200像素或更高
- **格式要求** - JPG或PNG格式
- **文件大小** - 不超过5MB
- **照片质量** - 清晰、正式、背景简洁

## ⌨️ 快捷键

- `Ctrl + S` - 保存简历
- `Ctrl + N` - 新建简历
- `Ctrl + O` - 打开简历管理
- `Ctrl + E` - 导出PDF

## 🔧 技术架构

### 前端技术
- **HTML5** - 现代化标记语言
- **CSS3** - 响应式设计和动画效果
- **JavaScript ES6+** - 模块化开发
- **Web APIs** - 文件处理和本地存储

### 核心模块
1. **FormHandler** - 表单数据处理和验证
2. **TemplateEngine** - 模板渲染引擎
3. **ExportManager** - 导出功能管理
4. **ResumeBuilderApp** - 主应用协调器

### 第三方库
- **SheetJS (xlsx.js)** - Excel文件处理
- **jsPDF** - PDF生成
- **html2canvas** - HTML转图片

## 📱 响应式设计

- **桌面端** - 完整功能体验
- **平板端** - 优化的触控界面
- **手机端** - 移动友好的布局
