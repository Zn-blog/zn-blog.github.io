# AI编程学习平台

一个交互式的AI编程学习网站，帮助你从零开始，循序渐进地学习AI相关编程。

## 功能特性

- 📚 **系统化课程**：11个完整的课程模块，从编程基础到AI实践
- 💻 **代码示例**：每个知识点都配有可运行的代码示例
- 📊 **进度跟踪**：自动保存学习进度，支持断点续学
- 🎨 **现代化UI**：美观的界面设计，良好的用户体验
- ✅ **完成标记**：标记已完成的课程，清晰了解学习进度

## 课程内容

### 基础课程（必学）

1. **编程基础入门** - 学习编程的基本概念和思维方式
2. **数据结构与算法基础** - 学习常用的数据结构和基本算法
3. **开发工具与环境配置** - 配置开发环境，掌握常用开发工具
4. **版本控制：Git基础** - 学习使用Git管理代码版本

### AI专业课程

5. **AI编程基础入门** - 了解AI的基本概念和发展历程
6. **Python基础回顾** - 掌握Python编程基础，为AI开发做准备
7. **机器学习基础** - 学习机器学习的基本概念和算法
8. **深度学习入门** - 了解神经网络和深度学习的基本原理
9. **自然语言处理（NLP）** - 学习如何处理和理解人类语言
10. **计算机视觉** - 学习如何让计算机"看见"和理解图像
11. **实践项目：构建AI应用** - 综合运用所学知识，构建完整的AI应用

## 如何使用

### 方法一：直接打开（推荐）

1. 直接在浏览器中打开 `index.html` 文件
2. 点击"开始学习"按钮
3. 按照课程顺序逐步学习

### 方法二：使用本地服务器

```bash
# 使用Python
python -m http.server 8000

# 使用Node.js
npx http-server

# 使用PHP
php -S localhost:8000
```

然后在浏览器中访问 `http://localhost:8000`

## 学习建议

1. **从基础开始**：如果你是编程新手，请务必先完成前4个基础课程
2. **按顺序学习**：课程设计是循序渐进的，建议按照顺序学习
3. **动手实践**：复制代码示例，在自己的环境中运行
4. **标记完成**：完成每个课程后记得标记，系统会记录你的进度
5. **反复复习**：可以随时返回之前的课程进行复习
6. **搭建环境**：在开始AI课程前，确保已经完成开发环境配置
