# 应用文件夹

这个文件夹用于存放小应用和小游戏的相关文件。

## 目录结构

```
apps/
├── README.md          # 说明文档
├── example-game/      # 示例游戏文件夹
│   └── index.html     # 游戏入口文件
└── example-tool/      # 示例工具文件夹
    └── index.html     # 工具入口文件
```

## 使用说明

1. 每个应用应该有独立的文件夹
2. 每个应用文件夹内应包含 index.html 作为入口文件
3. 应用的静态资源（图片、CSS、JS）应放在各自的文件夹内
4. 在后台管理系统中配置应用信息后，前台会自动展示
