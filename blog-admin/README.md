# 博客后台管理系统

一个功能完整的博客后台管理系统，采用现代化设计，支持文章、分类、标签、评论等全方位管理。

## 功能特性

### 🎯 核心功能
- **仪表盘** - 数据统计概览，最近文章和评论展示
- **文章管理** - 创建、编辑、删除文章，支持草稿和发布状态
- **分类管理** - 管理文章分类，查看每个分类的文章数量
- **标签管理** - 标签云展示，快速管理文章标签
- **评论管理** - 审核、回复、删除评论
- **媒体库** - 上传和管理图片等媒体文件
- **系统设置** - 网站基本信息配置

### ✨ 特色功能
- **Markdown 编辑器** - 支持 Markdown 语法的文章编辑器
- **实时预览** - 文章编辑时可实时预览效果
- **标签输入** - 便捷的标签输入和管理
- **图片上传** - 支持特色图片上传
- **响应式设计** - 完美适配各种设备屏幕

## 快速开始

### 1. 登录系统
打开 `login.html` 页面，使用以下凭据登录：
- **用户名**: admin
- **密码**: admin123

### 2. 主要页面

#### 仪表盘
- 显示网站统计数据（文章数、评论数、访问量等）
- 查看最近发布的文章
- 查看最新的评论

#### 文章管理
- 查看所有文章列表
- 点击"新建文章"创建新文章
- 编辑或删除现有文章
- 查看文章状态（已发布/草稿）

#### 分类管理
- 查看所有分类及其文章数量
- 新建、编辑、删除分类

#### 标签管理
- 以卡片形式展示所有标签
- 显示每个标签关联的文章数量
- 新建、编辑、删除标签

#### 评论管理
- 查看所有评论
- 审核待审核的评论
- 回复或删除评论

#### 媒体库
- 上传图片和其他媒体文件
- 管理已上传的媒体文件
- 删除不需要的文件

#### 系统设置
- 配置网站标题和描述
- 设置每页显示的文章数量
- 开启/关闭评论审核功能

### 3. 文章编辑器

打开 `editor.html` 使用文章编辑器：

#### 编辑功能
- **标题输入** - 在顶部输入文章标题
- **Markdown 编辑** - 使用 Markdown 语法编写文章
- **工具栏** - 快速插入常用 Markdown 格式
  - B - 加粗
  - I - 斜体
  - H - 标题
  - "" - 引用
  - { } - 代码
  - 🔗 - 链接
  - 🖼️ - 图片
  - 📋 - 列表

#### 文章设置
- **分类** - 选择文章所属分类
- **标签** - 添加文章标签（输入后按回车）
- **发布时间** - 设置文章发布时间
- **特色图片** - 上传文章封面图
- **摘要** - 编写文章摘要

#### 发布选项
- **发布文章** - 立即发布文章
- **保存草稿** - 保存为草稿，稍后发布
- **预览** - 预览文章效果

## 设计特点

### 🎨 视觉设计
- **宫崎骏风格配色** - 采用柔和的蓝绿色调
