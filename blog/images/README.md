# 背景图片说明

请将你的背景图片放在这个文件夹中，并命名为：

- `bg-1.jpg` - 第一张背景图片
- `bg-2.jpg` - 第二张背景图片
- `bg-3.jpg` - 第三张背景图片
- `bg-4.jpg` - 第四张背景图片

## 推荐规格

- 分辨率：1920x1080 或更高
- 格式：JPG、PNG 或 WebP
- 文件大小：建议压缩到 500KB 以内以提高加载速度

## 图片来源推荐

你可以从以下网站获取免费高质量图片：
- Unsplash (https://unsplash.com/)
- Pexels (https://pexels.com/)
- Pixabay (https://pixabay.com/)

## 注意事项

如果没有放置图片，系统会自动使用渐变色作为后备背景。
