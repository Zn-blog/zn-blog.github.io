## 📖 项目简介

这是一个功能完整的个人博客系统，包含**前台展示**和**后台管理**两部分。采用纯前端技术栈，支持 JSON 文件数据存储，无需数据库，开箱即用。

### ✨ 核心特点

- 🚀 **纯前端实现** - 无需复杂后端，快速部署
- 💾 **JSON 文件存储** - 数据持久化，支持备份和版本控制
- 🎨 **8种精美主题** - 蓝色海洋、紫色梦幻、绿色自然、橙色活力、深色模式、玻璃暗黑、粉色浪漫、赛博朋克
- 📱 **完全响应式** - 完美适配桌面端和移动端
- 🎵 **音乐播放器** - 网易云风格，支持歌词同步显示
- 📄 **飞书导入** - 一键导入飞书文档，自动处理图片
- 🔍 **智能搜索** - 多范围搜索，关键词高亮
- ⏰ **时光轴** - 三种视图（日历、列表、时间轴）
- 🌸 **樱花特效** - Canvas 渲染，可调速度
- 🖼️ **媒体库** - 统一管理图片、音乐、视频
- 🎮 **应用中心** - 内置小游戏和实用工具
- 🎬 **视频背景** - 动态视频背景，可自定义切换
- 🎯 **现代导航栏** - 滑动指示器，平滑动画效果

## 🚀 快速开始

### ⚡ 一键启动（推荐）

```bash
# 1. 克隆项目
git clone https://github.com/yourusername/personal-blog-system.git
cd personal-blog-system

# 2. 安装依赖
npm install

# 3. 启动项目
npm start
```

**就这么简单！** 启动脚本会自动：
- ✅ 检查Node.js版本（需要14+）
- ✅ 自动安装依赖
- ✅ 启动统一服务器
- ✅ 显示访问地址

### 📱 访问地址

启动成功后访问：
- 🏠 **博客首页**: http://localhost:3001/blog
- 🔧 **后台管理**: http://localhost:3001/blog-admin
- 🎮 **应用中心**: http://localhost:3001/blog/pages/apps.html
- 📡 **API接口**: http://localhost:3001/api

### 🛠️ 其他启动方式

```bash
# 开发模式（自动重启）
npm run dev

# 直接启动服务器
node unified-server.js

# 仅启动服务器
npm run server
```

## ❓ 常见问题

### Q: 数据会丢失吗？

A: 数据存储在 `data/` 目录的 JSON 文件中，不会丢失。建议定期备份。

### Q: 图片无法显示？

A: 检查以下几点：
1. 确保统一服务器正在运行
2. 检查图片路径是否正确
3. 查看浏览器控制台错误信息

### Q: 登录失败？

A: 尝试以下方法：
1. 使用默认账号：`admin` / `admin123`
2. 清除浏览器缓存
3. 检查 `data/users.json` 文件

### Q: 音乐无法播放？

A: 检查以下几点：
1. 确保音频文件格式正确（MP3、WAV、OGG）
2. 确保 URL 可访问
3. 查看浏览器控制台错误

### Q: 如何修改主题颜色？

A: 两种方式：
1. 使用内置的 8 种预设主题
2. 修改 `blog/css/themes.css` 中的 CSS 变量

### Q: 新建标签模态框显示错误标题？

A: 这是v2.0.1已修复的问题，解决方法：
1. **强制刷新页面**：按 `Ctrl+F5` 清除浏览器缓存
2. **检查控制台**：打开开发者工具查看调试信息
3. **使用测试文件**：打开 `test-tag-modal.html` 验证功能
4. **自动修正**：系统会自动检测并修正错误的标题显示

- **原因**：浏览器缓存、JavaScript执行顺序或模态框元素复用导致
- **解决方案**：多重保障机制确保标题正确显示
